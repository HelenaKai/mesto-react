# Проект: Mesto-React

### Версии проекта
Проектная работа по созданию интерактивного веб-сервиса на REACT, созданная в рамках прохождения десятого спринта курса Веб-разработчик в Яндекс.Практикум.

### Стек технологий:
* HTML 5
* CSS 3
* БЭМ Nested
* JavaScript
* React
* API
* Git
